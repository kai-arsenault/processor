# processor

The first part of this is the assembler. The assembler will take ARMv8 code and convert it to decimal and binary.

This is the comprehensive list of functions that can be processed:
- ADD
- SUB
- AND
- ORR
- EOR
- LSL
- LSR
- ADDI
- SUBI
- ANDI
- ORRI
- EORI
- SUBS
- SUBIS
- LDUR
- STUR
- CBZ
- CBNZ
- B
- B.EQ
- B.NE
- B.GT
- B.GTE
- B.LT
- B.LTE
